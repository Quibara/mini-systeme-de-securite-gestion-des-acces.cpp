# \# Système de Gestion des Accès Sécurisés (C++)

# 

# Ce projet est un laboratoire de sécurité informatique développé en C++. Il permet de gérer les accès utilisateurs de manière structurée.

# 

# \## 🛠️ Fonctionnalités

# \- Gestion des authentifications.

# \- Journalisation des accès (Logs).

# \- Interface console sécurisée.

# 

# \## 💻 Technologies

# \- Langage : C++

# \- Environnement : Visual Studio 2022

# \- Versioning : Git / GitHub

# 

# \## 🚀 Installation

# 1\. Cloner le dépôt : `git clone https://github.com/Quibara/mini-systeme-de-securite-gestion-des-acces.cpp.git`

# 2\. Ouvrir le fichier `.sln` dans Visual Studio.

# 3\. Compiler et exécuter.
